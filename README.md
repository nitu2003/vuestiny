# Vuestiny 命运2周报

Vuestiny 基于 Vue 开发，自动化显示本周的 __~~~RAID挑战，利维坦关卡顺序，巅峰星之塔和噬界者的武器限制等~~~__。

## 目前支持

- RAID 挑战
    - 深石地窖
    - 救赎花园
    - 最后遗愿
- 目的地
    - 木卫二
        - EXO挑战
        - DSC贮藏（痴狂强化）
    - 月球
        - 忧愁圣坛每日掉落
        - 徘徊着的梦魇
    - 千梦之城
        - 梦城污染程度
        - 梦城上维挑战位置
